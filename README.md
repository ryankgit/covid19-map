# covid19-map

Generates maps from COVID-19 data from https://github.com/GoogleCloudPlatform/covid-19-open-data


# Dependencies
~~~
pip install wheel
pip install numpy
pip install pandas
pip install shapely
pip install gdal
pip install fiona
pip install pyproj
pip install six
pip install rtree
pip install geopandas
pip install matplotlib
pip install descartes
pip install mapclassify
~~~
